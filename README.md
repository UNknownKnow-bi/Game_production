# 职场苏丹开发

一款模拟职场生存和晋升的互动剧情游戏，玩家需要在职场中做出各种选择，体验职场生活的方方面面。

## 游戏特点

- 引人入胜的剧情和角色
- 多结局选择系统
- 特权卡机制
- 动态适应的UI系统
- 高度自定义的文本显示系统

## 开发环境

- Godot 4.4
- GDScript

## 项目结构

- `assets/` - 游戏资源文件(图片、音频、字体等)
- `scenes/` - 游戏场景文件
- `scripts/` - 游戏脚本文件
- `backup/` - 备份文件

## 字体设置

游戏使用统一的中文字体 `assets/font/LEEEAFHEI-REGULAR.TTF`，通过自动加载的 FontManager 进行全局管理。
